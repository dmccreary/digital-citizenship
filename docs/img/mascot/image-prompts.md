# Maka the River Otter — Image Generation Prompts

These prompts produce consistent illustrations of **Maka the River Otter**, the
learning mascot for the *Digital Citizenship* textbook.

Maka is inspired by the river otters that live at **Bdote** — the confluence of
the Minnesota and Mississippi rivers, near ISD 197 in Minnesota.

## How to Use These Prompts

1. Copy one prompt at a time into your AI image generator (ChatGPT/DALL-E,
   Midjourney, Stable Diffusion, or similar).
2. Generate the image at **1024 × 1024 pixels** for best quality.
3. Save each image into `docs/img/mascot/` using the filename listed in the prompt.
4. After saving all 7 images, run the trim-padding script (see bottom of file)
   to remove excess transparent padding so the mascot displays at the right size.

Each prompt is **fully self-contained** — you do not need to copy a separate
"base description" first. Just paste one prompt at a time.

## After Generating Your Images

### Save Them to the Right Folder

All seven images go in:

```
docs/img/mascot/
├── neutral.png
├── welcome.png
├── thinking.png
├── tip.png
├── warning.png
├── encouraging.png
└── celebration.png
```

### Trim Excess Padding

AI image generators usually add a lot of transparent padding around the
character. Run the trim script on each image so Maka displays at the right
size in the admonition boxes:

```bash
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/neutral.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/welcome.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/thinking.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/tip.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/warning.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/encouraging.png
python ../claude-skills/src/image-utils/trim-padding-from-image.py docs/img/mascot/celebration.png
```
