# Digital Devices Explorer — Image Generation Prompt

Please generate a new image.

## Critical Rule

**This image must contain absolutely no text, labels, arrows, callout lines, numbers,
brand logos, or annotation marks of any kind.** All labeling will be added as an
interactive HTML overlay from a separate data file. Any embedded text will conflict
with the overlay system and must not appear. Do not include any letters, words,
numbers, or trademarked logos on any device screen, body, or background.

- **Format**: PNG
- **Dimensions**: 1200 × 900 px (landscape, 4:3 ratio)
- **Background**: Soft warm cream (#FBF6EC) with a faint river-blue (#2e6f8e)
  rounded shelf shape behind the devices, like a friendly classroom display table.
- **Style**: Modern flat cartoon vector illustration, clean rounded shapes,
  soft shadows, warm friendly palette with river-blue (#2e6f8e) accents.
  Suitable for a Grade 5 (ages 10–12) intelligent textbook.
  Generic, brand-free devices — no Apple, Samsung, Sony, Nintendo, Amazon, or
  any other identifiable brand. No logos, no app icons, no text on screens.
