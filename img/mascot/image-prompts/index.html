    Maka the River Otter — Image Generation Prompts

    These prompts produce consistent illustrations of Maka the River Otter, the learning mascot for the Digital Citizenship textbook.

    Maka is inspired by the river otters that live at Bdote — the confluence of the Minnesota and Mississippi rivers, near ISD 197 in Minnesota.

    How to Use These Prompts

    1. Copy one prompt at a time into your AI image generator (ChatGPT/DALL-E, Midjourney, Stable Diffusion, or similar).
    2. Generate the image at 1024 × 1024 pixels for best quality.
    3. Save each image into docs/img/mascot/ using the filename listed in the prompt.
    4. After saving all 7 images, run the trim-padding script (see bottom of file) to remove excess transparent padding so the mascot displays at the right size.

    Each prompt is fully self-contained — you do not need to copy a separate "base description" first. Just paste one prompt at a time.

    1. Neutral / Default Pose

    Filename: neutral.png

    I am about to ask you to generate 7 mascot images for a textbook. Please make them use a consistent style.

    Please generate a new neutral pose for Maka the River Otter. A modern flat cartoon vector illustration of Maka the River Otter, a friendly pedagogical mascot for a Grade 5 Digital Citizenship textbook. Maka has warm medium-brown fur with a cream-colored belly and chest, large kind dark eyes, small round whiskers, and a short rounded snout. Maka wears small round black glasses and a soft river-blue scarf (color #2e6f8e). Maka has a calm, friendly closed-mouth smile.

    Maka stands upright in a relaxed, neutral pose facing the viewer directly. Front paws rest naturally at the sides with no specific gesture. The pose is balanced and unassuming — suitable as a general-purpose or default illustration.

    Style: modern flat vector cartoon, clean lines, child-friendly, transparent background, suitable for embedding in educational content. No text in image.

    Please generate a new png image now with a fully transparent background now.
